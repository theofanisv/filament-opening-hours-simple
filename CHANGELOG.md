# Changelog

All notable changes to `filament-opening-hours` will be documented in this file.

## 1.0.0 - 2025-12-21

### Added
- Initial release
- OpeningHoursInput form component for editing opening hours
- OpeningHoursEntry infolist component for displaying opening hours
- OpeningHoursColumn table column for compact display
- Support for day-by-day hour editing (Monday-Sunday)
- Multiple time ranges per day support
- Exceptions for specific dates
- Overflow (midnight crossing) support
- Comprehensive real-time validation
- Full integration with spatie/opening-hours v4
- Translation support
- Configuration file for customization
